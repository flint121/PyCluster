# PyCluster
PyCluster is a standalone program for clustering built in Python. It is also planned to be published as a package for data analytics in Python. 
PyCluster allows clustering of data pulled from CSV files using various similarity metrics and clustering algorithms. It also includes the functionality to provide useful summaries and analytics about this data in a format easily understood by the user. PyCluster can be used as a standalone program currently and is in the process of being published as a Python package.
